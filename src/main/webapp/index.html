Notion de terminal operation

Le stream pipeline est lazy : les calculs seulement réalisés lors de la terminal operation


int sum = widgets.stream()
    .filter(b -> b.getColor() == RED)
    .mapToInt(b -> b.getWeight())
    .sum();
